机器学习概述

基本概念

机器学习 (Machine Learning, ML) 是人工智能的一个核心子领域。它的本质，是通过计算机算法自动地从数据中学习其内在的规律，并利用这些学到的规律对全新的、未知的数据进行预测或分类。
机器学习算法的一个关键特点是，它并非依赖于人类预先编写的固定规则，而是从提供的样本数据 (Sample Data)中自行分析并获得某种规律，再利用这个规律对未知数据进行预测。这使得机器学习成为一门涉及概率论、统计学、矩阵论、优化理论、算法复杂度理论等多个领域的交叉学科。
机器学习特别适合处理具有以下特征的问题：
  • 大数据：当数据量极其庞大，人类专家难以凭经验和直觉进行有效分析时，机器学习可以高效地从中发现模式。
  • 黑箱或复杂系统：对于一些内部机理极其复杂、难以找到精确的控制方程 (Governing Equations) 来描述的系统（如金融市场预测、自然语言理解），机器学习可以通过学习输入和输出之间的关系来进行建模。

主要类别

机器学习可以根据训练数据是否带有“答案”或“标签”，分为以下两大主要类别：
  • 有监督学习 (Supervised Learning)：这是最常见的一种学习方式。算法的训练数据是有标签 (Labeled) 的，意味着每个数据样本都配有一个我们已知的、正确的输出结果（即标签）。模型的目标就是学习从输入数据到输出标签之间的映射关系，然后用这个模型去推断新样本的标签。
  • 无监督学习 (Unsupervised Learning)：在这种情况下，训练数据是没有标签 (Unlabeled) 的。算法需要自己去探索和发现数据内部隐藏的结构、模式或分布。
此外，还有一种结合了两者的半监督学习 (Semi-supervised Learning)，它使用少量有标签数据和大量无标签数据进行学习，在现实中也非常有用。
四大类核心算法
根据数据是否有标签，以及标签的类型，机器学习算法可以被归纳为四个主要类别：
  1. 回归 (Regression) - 有监督学习： 当输出的标签是连续的数值时，我们称之为回归问题。例如，预测明天的气温、预测一个房屋的价格、或者预测一只股票的未来价值。其目标是预测一个具体的数值。
  2. 分类 (Classification) - 有监督学习： 当输出的标签是离散的类别时，我们称之为分类问题。例如，判断一封邮件是否是垃圾邮件（是/否）、识别一张图片中的动物是猫还是狗、或者决定是否批准一笔贷款（批准/不批准）。其目标是将数据划分到预定义的类别中。
  3. 聚类 (Clustering) - 无监督学习： 当数据没有标签，而我们的目标是根据数据样本自身的特征，将它们自动地分成不同的组（或称为“簇”）时，这就是聚类问题。其核心思想是让同一组内的数据尽可能相似，而不同组之间的数据尽可能不同。例如，根据用户的购买行为将客户分群，以便进行精准营销。
  4. 降维 (Dimensionality Reduction) - 无监督学习： 当数据的特征（维度）非常多时，会给数据处理和建模带来困难。降维的目标是在尽可能保留原始数据信息的前提下，减少特征的数量。这有助于简化模型、节省计算资源，并可能去除数据中的噪声。主成分分析 (Principal Component Analysis, PCA) 是最经典的降维算法之一。

监督学习

信用卡例子

以信用卡审批为例，银行或金融机构每天都会收到成百上千的信用卡申请。传统上，这个过程需要人工审核，由信贷员根据经验和一系列规则来判断。但这种方式效率低下，成本高昂，且人工决策可能受到主观或无意识偏见的影响。
我们希望创建一个自动化的、高效的、客观一致的决策系统。这个系统能够学习历史上成功和失败的案例，并基于数据做出快速且可靠的判断，从而大大提升审批效率和决策质量。
首先，需要把这个现实世界的问题，转化成一个机器可以理解的、有明确输入和输出的数学问题。
  • 任务 (Task)：预测一个申请人是否应该被批准信用卡。这是一个非常明确的商业问题。
  • 输入 (Applicant's info)：为了做出这个判断，我们需要申请人的一些信息，比如年龄（age）、性别（gender）、薪水（salary）、国籍（citizenship）、工作年限（years in job）等等。这些信息就是我们做决策的依据。
  • 目标：我们的目标是创建一个自动化系统（也就是机器学习模型），这个系统能接收申请人的信息作为输入，然后自动输出一个决策：”批准“ 还是 ”不批准“。
这里的任务在机器学习中被称为二元分类 (Binary Classification)，因为最终的输出结果只有两种可能（批准/不批准）。
我们需要找到大量过去已经处理过的信用卡申请案例。由于计算机和数学模型更擅长处理数字，将原始数据（如性别“male/female”、国籍“CN”）预处理成数值形式。
  • 样本 (Sample)：表格中的每一行代表一个独立的申请案例，也就是一个样本。例如，第一行[2.5, 1, 10, 3, 1]就是关于“申请人1”的所有信息。在数学上，我们通常用xix_ixi​来表示第ii i个样本。
  • 特征 (Feature)：表格中的每一列代表了样本的一个特定属性，比如“年龄”、“性别”、“薪水”等。这些就是特征。特征是我们用来描述一个样本的各个维度。
  • 特征矩阵 (Feature Matrix)：当我们把所有样本的特征组织在一起，就形成了一个矩阵。这个矩阵通常用大写的X\boldsymbol{X}X来表示。在这个矩阵里，每一行xi⊤\boldsymbol{x}_{i}^{\top}xi⊤​是一个完整的样本，每一列是一个特定的特征。

  • 标签 (Label)：标签就是每个历史样本对应的真实结果。在这里，“approve”这一列就是标签。+1+1+1代表“批准”，−1-1−1代表“不批准”。在数学上，我们通常用yiy_iyi​来表示第ii i个样本的标签。
  • 标签向量 (Label Vector)y：把所有样本的标签集合在一起，就形成了一个向量，通常用小写的yy y表示。
  • 监督 (Supervision)：因为我们为机器提供了带有正确答案（标签）的学习材料（特征），所以这个过程被称为监督学习。

组成部分的定义

数据 (Data)

数据是学习的基础，其形式和空间定义了问题的边界。
  • 输入空间 (Input Space) / 特征空间 (Feature Space) X\mathcal{X}X
    这是所有可能输入的集合。每一个输入，也称为一个实例 (Instance) 或样本 (Sample)，通常由一个特征向量 (Feature Vector) x\boldsymbol{x}x来表示。
    x=(x(1),x(2),…,x(d))\boldsymbol{x}=(x^{(1)},x^{(2)},\ldots,x^{(d)})x=(x(1),x(2),…,x(d))
    其中d d d是特征的数量或维度 (dimension)。特征空间X\mathcal{X}X通常是d d d维欧几里得空间Rd\mathbb{R}^{d}Rd的一个子集。
  • 输出空间 (Output Space) / 标签空间 (Label Space) Y\mathcal{Y}Y
    这是所有可能输出的集合。根据输出空间的性质，监督学习任务被分为两类：
    • 分类 (Classification)：如果输出空间Y\mathcal{Y}Y是一个有限的、离散的集合，例如Y={+1,−1}\mathcal{Y} = \{+1, -1\}Y={+1,−1}或 Y={猫, 狗, 鸟}\mathcal{Y} = \{\text{猫, 狗, 鸟}\}Y={猫, 狗, 鸟}，则该任务为分类任务。
    • 回归 (Regression)：如果输出空间Y\mathcal{Y}Y是一个连续的数值集合，例如所有实数R\mathbb{R}R或某个区间[a,b][a, b][a,b]，则该任务为回归任务。
  • 目标函数 (Target Function) ggg
    我们假设在数据背后存在一个潜在的、未知的真实映射规律，记为目标函数ggg。
    g:X→Yg:\mathcal{X}\rightarrow\mathcal{Y}g:X→Y
    这个ggg是我们学习的终极目标，它完美地描述了输入与输出之间的关系。在实际问题中，gg g是无法被直接获知的（例如，信用卡审批背后那个最完美的决策模型）。
  • 训练数据集 (Training Dataset) D\mathcal{D}D
    这是学习算法可用的“经验”。它是一个包含了NNN个独立同分布 (i.i.d.) 的样本对的集合：
    D={(x1,y1),(x2,y2),…,(xN,yN)}\mathcal{D} = \{(\boldsymbol{x}_1, y_1), (\boldsymbol{x}_2, y_2), \ldots, (\boldsymbol{x}_N, y_N)\}D={(x1​,y1​),(x2​,y2​),…,(xN​,yN​)}
    其中，每个(xi,yi)(\boldsymbol{x}_i, y_i)(xi​,yi​)都是一个有标签的样本 (Labeled Instance)，满足xi∈X\boldsymbol{x}_i \in \mathcal{X}xi​∈X 且 yi∈Yy_i \in \mathcal{Y}yi​∈Y。我们假定存在一个未知的、真实的联合概率分布P(X,Y)P(\mathcal{X}, \mathcal{Y})P(X,Y)，而训练集D\mathcal{D}D是从这个分布中采样得到的。

模型 (Model)

模型是学习的目标，它代表了我们希望从数据中学到的规律。
  • 假设空间 (Hypothesis Space) H\mathcal{H}H
    在学习开始之前，我们需要预先确定 (pre-determined) 一个模型的“候选范围”，这个范围就是假设空间H\mathcal{H}H。它是由所有可能被我们学习算法考虑的函数构成的集合。H\mathcal{H}H的选择决定了模型的类型：
    • 线性假设空间：如果H\mathcal{H}H是所有线性函数的集合，那么我们最终会得到线性模型（如线性回归、线性分类）。
    • 非线性假设空间：如果H\mathcal{H}H包含非线性函数，例如所有可能的神经网络结构，我们就会得到非线性模型。
  • 假设/模型 (Hypothesis/Model) fθf_{\boldsymbol{\theta}}fθ​
    假设空间H\mathcal{H}H中的每一个具体的候选函数，就是一个假设或模型。为了能够通过算法进行学习和优化，这个函数通常被一组参数 (parameters) θ\thetaθ所定义，因此我们将其记为fθf_{\theta}fθ​。
    fθ∈Hf_{\boldsymbol{\theta}} \in \mathcal{H}fθ​∈H
    • 线性回归中, fθ(x)=θ⊤xf_{\boldsymbol{\theta}}(\boldsymbol{x})=\boldsymbol{\theta}^\top xfθ​(x)=θ⊤x。这里的参数θ\boldsymbol{\theta}θ就是模型的权重向量。
    • 深度学习中, fθf_{\boldsymbol{\theta}}fθ​就是整个神经网络，参数θ\boldsymbol{\theta}θ代表了网络中所有的权重 (weights) 和偏置 (biases)。
我们的目标，就是从假设空间H\mathcal{H}H中，通过调整参数θ\boldsymbol{\theta}θ，找到一个能最好地近似目标函数ggg的模型fθf_{\boldsymbol{\theta}}fθ​：
fθ≈gf_{\boldsymbol{\theta}} \approx gfθ​≈g

学习策略 (Learning Strategy)

学习策略定义了“如何”从假设空间H\mathcal{H}H中挑选出最优模型。它包含两个关键部分：损失函数和优化算法。
  • 损失函数 (Loss Function) ℓ\ellℓ
    损失函数（也称为成本函数 Cost Function 或目标函数 Objective Function）是一个用于量化模型预测错误程度的函数。它衡量的是单个样本的真实标签yiy_i yi​与模型的预测值fθ(xi)f_{\boldsymbol{\theta}}(\boldsymbol{x}_i)fθ​(xi​)之间的差异。
    ℓ(fθ(xi),yi)\ell (f_{\boldsymbol{\theta}}(\boldsymbol{x}_i), y_i )ℓ(fθ​(xi​),yi​)
    • 对于回归问题，常用的损失函数是平方损失 (Squared Loss): ℓ(fθ(xi),yi)=(yi−fθ(xi))2\ell (f_{\boldsymbol{\theta}}(\boldsymbol{x}_i), y_i ) = (y_i - f_{\boldsymbol{\theta}}(\boldsymbol{x}_i))^2ℓ(fθ​(xi​),yi​)=(yi​−fθ​(xi​))2。
    • 对于分类问题，常用的是0-1损失 (0-1 Loss)：如果y≠f(x)y \neq f(x)y=f(x)，ℓ(fθ(xi),yi)=1\ell (f_{\boldsymbol{\theta}}(\boldsymbol{x}_i), y_i ) = 1ℓ(fθ​(xi​),yi​)=1，否则为0 00，以及交叉熵损失 (Cross-Entropy Loss)。
  • 最优化问题 (Learning/Optimization Problem)
    学习的核心过程, 就是寻找一组最优参数θ^\hat{\boldsymbol{\theta}}θ^, 使得模型在整个训练集上的平均损失最小化。这个过程可以被严格地表述为以下最优化问题 (P)：
    θ^=argmin⁡θ∈Rd1n∑i=1nℓ(fθ(xi),yi)\hat{\boldsymbol{\theta}} = \operatorname*{argmin}_{\boldsymbol{\theta}\in \mathbb{R}^{d}} \frac{1}{n} \sum_{i=1}^{n} \ell(f_{\boldsymbol{\theta}}(\boldsymbol{x}_i), y_i)θ^=θ∈Rdargmin​n1​i=1∑n​ℓ(fθ​(xi​),yi​)
    这个公式的含义是: 寻找能让训练集上总损失的均值最小化的那组参数θ\boldsymbol{\theta}θ，并将这组最优参数记为θ^\hat{\boldsymbol{\theta}}θ^。
  • 优化算法 (Optimization Algorithm) A\mathcal{A}A
    一个具体的优化算法A\mathcal{A}A (例如梯度下降法) 被设计用来求解上述的最优化问题 (P)。算法A\mathcal{A}A通过系统性的方法调整参数θ\boldsymbol{\theta}θ，以逐步降低损失函数的值，最终找到最优解θ^\hat{\boldsymbol{\theta}}θ^。

PCA的数学模型

当我们建立了主成分分析 (Principal Component Analysis, PCA) 的几何直觉，就可以深入探讨它的数学建模及其目标。

构建数学模型

主成分分析 (PCA) 的核心思想是为高维数据（ddd维）找到一个最优的低维近似（kkk维）。这个“最优”是通过最小化数据点与其在低维空间近似点之间的距离（的平方范数）来定义的。

数据点的低维表示

为了实现这一点，我们构建了如下的数学模型，它描述了原始数据点xi\boldsymbol{x}_ixi​如何通过低维表示θi\boldsymbol{\theta}_iθi​来近似重建：
xi≈Aθi+μ\boldsymbol{x}_i \approx \boldsymbol{A}\boldsymbol{\theta}_i + \boldsymbol{\mu}xi​≈Aθi​+μ
我们来详细分解这个模型中的每一个组成部分：
  1. xi∈Rd\boldsymbol{x}_i \in \mathbb{R}^dxi​∈Rd：这是我们的原始样本，一个位于ddd维空间中的向量。
  2. μ∈Rd\boldsymbol{\mu} \in \mathbb{R}^dμ∈Rd：这是一个未知的偏移向量 (offset vector)。在几何上，它定义了我们kkk维子空间的位置（即这个“平面”的截距）。
  3. A∈Rd×k\boldsymbol{A} \in \mathbb{R}^{d \times k}A∈Rd×k：这是一个d×kd \times kd×k维的矩阵，它通常被称为基 (basis)。这个矩阵至关重要，它的kkk个列向量（每个都是ddd维的）定义了kkk维子空间的“朝向”或“方向”。
    1. 模型要求A\boldsymbol{A}A的列是标准正交 (orthogonal columns) 的，即满足A⊤A=Ik\boldsymbol{A}^\top \boldsymbol{A} = \boldsymbol{I}_kA⊤A=Ik​（单位矩阵）。
    2. 这确保我们的kkk个基向量是互相垂直的单位向量，能构成该子空间的一组标准基底。
  4. θi∈Rk\boldsymbol{\theta}_i \in \mathbb{R}^kθi​∈Rk：这是xi\boldsymbol{x}_ixi​在kkk维子空间中的低维坐标。我们的目标k≪dk \ll dk≪d就是为了得到这个压缩后的表示。
解释：这个公式xi≈x^i=Aθi+μ\boldsymbol{x}_i \approx \hat{\boldsymbol{x}}_i = \boldsymbol{A}\boldsymbol{\theta}_i + \boldsymbol{\mu}xi​≈x^i​=Aθi​+μ描述的是一个仿射子空间 (Affine Subspace)。以上图为例，PCA将3D空间中的数据点xi\boldsymbol{x}_ixi​映射为2D空间中的x^i\hat{\boldsymbol{x}}_ix^i​。
  • 线性子空间：矩阵A\boldsymbol{A}A的列向量a1,a2\boldsymbol{a}_1,\boldsymbol{a}_2a1​,a2​张成k=2k=2k=2维线性子空间（图中底部平面）。a1,a2\boldsymbol{a}_1,\boldsymbol{a}_2a1​,a2​是标准正交的（互相垂直且长度为1）。这个平面穿过原点。
  • 平移（μ\boldsymbol{\mu}μ）：黄色向量是我们模型的偏移向量 (offset)。它代表了我们“最佳”2D平面（图中顶部平面）的“中心点”相对于原点的位置。
  • 仿射子空间：将底部的“原点平面”整体平移μ\boldsymbol{\mu}μ的距离得到最终的仿射子空间。它具有与底部平面完全相同的“朝向”（由A\boldsymbol{A}A定义），但具有不同的“位置”（由μ\boldsymbol{\mu}μ定义）。
  • 重建点x^i\hat{\boldsymbol{x}}_ix^i​：绿色向量是我们对xi\boldsymbol{x}_ixi​的重建点。根据x^i=Aθi+μ\hat{\boldsymbol{x}}_i = \boldsymbol{A}\boldsymbol{\theta}_i + \boldsymbol{\mu}x^i​=Aθi​+μ，它位于顶部的仿射子空间上。它在仿射子空间中的坐标是θi=[θi,1,θi,2]⊤\boldsymbol{\theta}_i = [\theta_{i,1}, \theta_{i,2}]^\topθi​=[θi,1​,θi,2​]⊤。
  • 原始数据点xi\boldsymbol{x}_ixi​：红点是我们想要近似的原始数据点，红色虚线代表了重构误差，即原始点xi\boldsymbol{x}_ixi​与它在“最佳”平面上的重建点x^i\hat{\boldsymbol{x}}_ix^i​之间的几何距离。

PCA的目标函数

基于上述模型，PCA的学习问题就转变为一个最优化问题。我们的目标是找到最佳的均值μ\boldsymbol{\mu}μ、最佳的基A\boldsymbol{A}A以及所有数据点对应的最佳低维坐标{θi}\{\boldsymbol{\theta}_i\}{θi​}，使得原始点xi\boldsymbol{x}_ixi​和它的重建点Aθi+μ\boldsymbol{A}\boldsymbol{\theta}_i + \boldsymbol{\mu}Aθi​+μ之间的平均重构误差 (reconstruction error) 最小。
这个目标函数（即PCA的代价函数）可以写为：
minimizeμ,{θi},A⊤A=Ik1n∑i=1n∥xi−Aθi−μ∥22\underset{\boldsymbol{\mu}, \{\boldsymbol{\theta}_i\}, \boldsymbol{A}^\top \boldsymbol{A} = \boldsymbol{I}_k}{\text{minimize}} \quad \frac{1}{n} \sum_{i=1}^n \|\boldsymbol{x}_i - \boldsymbol{A}\boldsymbol{\theta}_i - \boldsymbol{\mu}\|_2^2μ,{θi​},A⊤A=Ik​minimize​n1​i=1∑n​∥xi​−Aθi​−μ∥22​
这里：
  • ∥xi−Aθi−μ∥22\|\boldsymbol{x}_i - \boldsymbol{A}\boldsymbol{\theta}_i - \boldsymbol{\mu}\|_2^2∥xi​−Aθi​−μ∥22​计算的是中心化后的单个数据点xi−μ\boldsymbol{x}_i - \boldsymbol{\mu}xi​−μ与其重建点Aθi\boldsymbol{A}\boldsymbol{\theta}_iAθi​之间的平方欧氏距离。
  • 1n∑i=1n…\frac{1}{n} \sum_{i=1}^n \ldotsn1​∑i=1n​…是对所有nnn个样本的误差求平均，即最小化均方误差 (MSE)。
  • A⊤A=Ik\boldsymbol{A}^\top \boldsymbol{A} = \boldsymbol{I}_kA⊤A=Ik​是我们对基矩阵A\boldsymbol{A}A施加的标准正交约束，即要求它的kkk个列向量互相垂直。
综上所述，PCA的真正目标是：找到一个最佳的kkk维“仿射子空间”（一个由A\boldsymbol{A}A定义朝向、由μ\boldsymbol{\mu}μ定义位置的“平面”），以及每个数据点在这个平面上的最佳坐标{θi}\{\boldsymbol{\theta}_i\}{θi​}，使得所有原始点xi\boldsymbol{x}_ixi​和它们的重建点x^i\hat{\boldsymbol{x}}_ix^i​之间的总距离（误差）最小。
这个优化问题存在一些特点：
  1. 已知与未知：在这个问题中，我们唯一已知的是nnn个原始样本{xi}\{\boldsymbol{x}_i\}{xi​}和对应的。而偏移向量μ\boldsymbol{\mu}μ、基矩阵A\boldsymbol{A}A以及所有nnn个低维表示{θi}\{\boldsymbol{\theta}_i\}{θi​}都是需要求解的未知参数。
  2. 非凸问题 (Nonconvex optimization)：由于目标函数中存在未知变量A\boldsymbol{A}A和θi\boldsymbol{\theta}_iθi​的乘积项（Aθi\boldsymbol{A}\boldsymbol{\theta}_iAθi​），这个优化问题在同时求解A\boldsymbol{A}A和θi\boldsymbol{\theta}_iθi​时是一个非凸问题，这通常会使求解变得困难。
  3. 难点在于求解A\boldsymbol{A}A：整个问题的核心和难点在于找到最优的基A\boldsymbol{A}A。
    给定A\boldsymbol{A}A则问题简化：如果基A\boldsymbol{A}A是已知的（即我们已经知道要投影到哪个子空间），那么求解μ\boldsymbol{\mu}μ和{θi}\{\boldsymbol{\theta}_i\}{θi​}将变得非常简单：
    μ\boldsymbol{\mu}μ就是样本均值。对于每个θi\boldsymbol{\theta}_iθi​，问题变为一个简单的最小二乘问题，其闭式解（即投影）为θi=A⊤(xi−μ)\boldsymbol{\theta}_i = \boldsymbol{A}^\top (\boldsymbol{x}_i - \boldsymbol{\mu})θi​=A⊤(xi​−μ)。
因此，PCA算法的核心就在于如何利用数据的统计特性（如协方差矩阵）来高效地解出这个“最难的部分”——基矩阵A\boldsymbol{A}A。

形式1：最小化重构误差

回顾一下，我们的原始目标函数是：
minimizeμ,{θi},A⊤A=Ik1n∑i=1n∥xi−Aθi−μ∥22\underset{\boldsymbol{\mu}, \{\boldsymbol{\theta}_i\}, \boldsymbol{A}^\top \boldsymbol{A} = \boldsymbol{I}_k}{\text{minimize}} \quad \frac{1}{n} \sum_{i=1}^n \|\boldsymbol{x}_i - \boldsymbol{A}\boldsymbol{\theta}_i - \boldsymbol{\mu}\|_2^2μ,{θi​},A⊤A=Ik​minimize​n1​i=1∑n​∥xi​−Aθi​−μ∥22​
这个优化问题同时求解μ\boldsymbol{\mu}μ,{θi}\{\boldsymbol{\theta}_i\}{θi​}和A\boldsymbol{A}A是困难的。因此，我们采用的策略是：固定其中两组变量，求解另一组变量，即通过交替求解 (alternating optimization) 来逐步解决PCA目标函数的过程。

1.求解θi\boldsymbol{\theta}_iθi​ (假设A\boldsymbol{A}A和μ\boldsymbol{\mu}μ已知)

首先，假设基矩阵A\boldsymbol{A}A和均值向量μ\boldsymbol{\mu}μ已经是固定的。我们需要为每个数据点iii找到其最佳的低维表示θi\boldsymbol{\theta}_iθi​。
我们的目标函数（省略了常数1n\frac{1}{n}n1​）变为：
minimize{θi}∑i=1n∥xi−Aθi−μ∥22\underset{\{\boldsymbol{\theta}_i\}}{\text{minimize}} \sum_{i=1}^n \|\boldsymbol{x}_i - \boldsymbol{A}\boldsymbol{\theta}_i - \boldsymbol{\mu}\|_2^2{θi​}minimize​i=1∑n​∥xi​−Aθi​−μ∥22​
一个关键的观察是，θi\boldsymbol{\theta}_iθi​只出现在求和i=1i=1i=1到nnn中的第iii项。因此，最小化这个总和等价于对每一个iii独立地最小化其对应的项。
这意味着我们将一个大的优化问题分解成了nnn个独立的最小二乘法 (Least Squares) 问题：
minimizeθi∥(xi−μ)−Aθi∥22for i=1,…,n\underset{\boldsymbol{\theta}_i}{\text{minimize}} \quad \|(\boldsymbol{x}_i - \boldsymbol{\mu}) - \boldsymbol{A}\boldsymbol{\theta}_i\|_2^2 \quad \text{for } i = 1, \ldots, nθi​minimize​∥(xi​−μ)−Aθi​∥22​for i=1,…,n
这是一个标准形式min⁡θ∥y−Xθ∥22\min_{\boldsymbol{\theta}} \|\boldsymbol{y} - \boldsymbol{X}\boldsymbol{\theta}\|_2^2minθ​∥y−Xθ∥22​的问题，其中y=(xi−μ)\boldsymbol{y} = (\boldsymbol{x}_i - \boldsymbol{\mu})y=(xi​−μ)是“目标标签”（中心化数据），X=A\boldsymbol{X} = \boldsymbol{A}X=A是“数据矩阵”，θ=θi\boldsymbol{\theta} = \boldsymbol{\theta}_iθ=θi​是我们要找的"权重"。
根据最小二乘法，其标准解为：
θi=(A⊤A)−1A⊤(xi−μ)\boldsymbol{\theta}_i = (\boldsymbol{A}^\top \boldsymbol{A})^{-1} \boldsymbol{A}^\top (\boldsymbol{x}_i - \boldsymbol{\mu})θi​=(A⊤A)−1A⊤(xi​−μ)
根据我们对A\boldsymbol{A}A的约束A⊤A=Ik\boldsymbol{A}^\top \boldsymbol{A} = \boldsymbol{I}_kA⊤A=Ik​（k×kk \times kk×k单位矩阵），上式可以立即简化：
θi=(Ik)−1A⊤(xi−μ)=A⊤(xi−μ)\boldsymbol{\theta}_i = (\boldsymbol{I}_k)^{-1} \boldsymbol{A}^\top (\boldsymbol{x}_i - \boldsymbol{\mu}) = \boldsymbol{A}^\top (\boldsymbol{x}_i - \boldsymbol{\mu})θi​=(Ik​)−1A⊤(xi​−μ)=A⊤(xi​−μ)
解释
这个公式是PCA中非常重要的“编码”操作。为简单起见，我们设偏移后的向量xc=(xi−μ)\boldsymbol{x}_c = (\boldsymbol{x}_i - \boldsymbol{\mu})xc​=(xi​−μ)。那么：

A\boldsymbol{A}A是一个d×kd \times kd×k矩阵（高×\times×窄），它的列是kkk个ddd维的标准正交基向量（即主成分方向）。

所以，θi=[θ1,θ2,…,θk]⊤\boldsymbol{\theta}_i = [\theta_1, \theta_2, \ldots, \theta_k]^\topθi​=[θ1​,θ2​,…,θk​]⊤这个向量，就是xc\boldsymbol{x}_cxc​在kkk个新的、正交的坐标轴a1,…,ak\boldsymbol{a}_1, \ldots, \boldsymbol{a}_ka1​,…,ak​上的投影值。

3.求解A\boldsymbol{A}A

现在，我们已经确定了最优的θi\boldsymbol{\theta}_iθi​和μ\boldsymbol{\mu}μ（分别由A,μ\boldsymbol{A}, \boldsymbol{\mu}A,μ和数据xi\boldsymbol{x}_ixi​决定）。我们可以将它们代入，以得到一个只关于A\boldsymbol{A}A的最终优化问题。
我们采取一个非常重要的简化步骤：数据中心化 (Centering)。既然我们知道最优的μ\boldsymbol{\mu}μ就是样本均值xˉ\bar{\boldsymbol{x}}xˉ，我们可以首先从所有数据中减去这个均值：
Setxi←xi−μ\text{Set} \quad \boldsymbol{x}_i \leftarrow \boldsymbol{x}_i - \boldsymbol{\mu}Setxi​←xi​−μ
在不失一般性的情况下，我们可以假设数据已经中心化，即μ=0\boldsymbol{\mu} = \boldsymbol{0}μ=0。
当μ=0\boldsymbol{\mu} = \boldsymbol{0}μ=0时：
  1. 最优的θi\boldsymbol{\theta}_iθi​变为θi=A⊤xi\boldsymbol{\theta}_i = \boldsymbol{A}^\top \boldsymbol{x}_iθi​=A⊤xi​。
  2. 原始目标函数中的μ\boldsymbol{\mu}μ变为0\boldsymbol{0}0。
将这两点代入原始目标函数：

3.得到第三种形式

将S\boldsymbol{S}S代入，我们得到了PCA的第三种形式，也是最终的优化问题：
maximizeA⊤A=Iktrace(A⊤SA)\underset{\boldsymbol{A}^\top \boldsymbol{A} = \boldsymbol{I}_k}{\text{maximize}} \quad \text{trace}(\boldsymbol{A}^\top \boldsymbol{S} \boldsymbol{A})A⊤A=Ik​maximize​trace(A⊤SA)
